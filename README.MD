# Pitch App

The project is about a one pitch website.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install python dependencies.

## Requirements/Dependencies

* Flask==1.0.2
* Flask-Bootstrap4==4.0.2
* Flask-Script==2.0.6
* gunicorn==19.9.0
* Jinja2==2.10.1
* requests==2.21.0
* urllib3==1.24.1
* Werkzeug==0.15.2
